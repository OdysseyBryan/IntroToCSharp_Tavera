**Codac Logistics Delivery & Fuel Auditor**


**Description**
A CLI or Console-Based fuel tracking and delivery performance auditing tool programmed for Codac Logistics. It helps companies with delivery services to monitor daily fuel expenses, calculate delivery efficiency, and generatee professional audit reports for accounting.


**Why it is useful:**
- It tracks a 5-day fuel expenses for delivery vehicles
- It calculates fuel efficiency ratios such as Kilometers to PHP (Philippine Peso)
- Validate driver inputs to ensure data accuracy
- Generate professional audit reports for management
- Compares performance to all multiple drivers (if applicable)


**Problem it solves:**
Codac Logistics needed a simple but effective tool to monitor their delivery fuel consumption and efficiency. Manual tracking was error-prone and time consuming. This automated solution provide an accurate calculations and reporting.

**Intallation**

**Prerequisites**
1. Visual Studio 2019 or later (or any compatible IDE)
2. .NET Framework 4.7.2 or higher
3. Git (for version control)

**Setup**
1. Clone the repository on your CMD with your chosen folder and paste this prompt:
git clone https://github.com/OdysseyBryan/IntroToCSharp_Tavera.git
cd IntroToCSharp_Tavera

2. Open in Visual Studio:
- Open CodacLogisticsAuditor.sln in Visual Studio
- Build the solution

3. Run the application:
- Click the "Start" button
- And wait until the application starts


**Basic Usage**
1. Run the program
2. Enter driver information:
- Full name
- Weekly fuel budget in PHP
- Total distance travelled from 1.0 to 5000.0
3. Enter daily expenses for 5 days:
- Fuel cost for each day
- Distance travelled each day
4. View the audit report:
- Daily breakdown table
- Efficiency calculations
- Budget status
- Visual trend analysis

**Advanced Features**
- Multi-driver comparison: by entering up to 10 drivers to compare their efficiency
- Color-coded ratings
- Visual indicators such as Bar charts and trend arrows
- Error handling: Input validation with error messages
